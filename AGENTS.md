## Project Summary
Tech House e-commerce platformasi uy jihozlari va maishiy texnikalar sotuvi bilan shug'ullanadi. Loyiha junior developer darajasiga mos ravishda, sodda va tushunarli kodlar bilan yaratilgan.

## Tech Stack
- HTML5
- CSS3 (Custom styles)
- JavaScript (Minimal vanilla JS)
- Remix Icon (Ikonkalar uchun)

## Architecture
- `index.html`: Bosh sahifa
- `about.html`: Kompaniya haqida ma'lumot
- `blog.html`: Yangiliklar va maqolalar
- `contact.html`: Bog'lanish ma'lumotlari va forma
- `product_detail.html`: Mahsulot tafsilotlari
- `cart.html`: Savat sahifasi
- `checkout.html`: Buyurtmani rasmiylashtirish prototipi
- `assets/css/styles.css`: Barcha uslublar
- `assets/js/script.js`: Interaktiv elementlar (rasm almashtirish, h.k.)

## User Preferences
- Kodlar junior developer darajasida, sodda yozilgan.
- API larsiz, faqat HTML fayllararo navigatsiya ishlatilgan.
- UI dizayni Figma asosida professional va modern holatga keltirilgan.

## Project Guidelines
- Barcha sahifalar responsive (mobil qurilmalarga mos).
- JS kodlari maksimal darajada sodda va tushunarli.
- Ikonkalar uchun Remix Icon kutubxonasidan foydalanilgan.

## Common Patterns
- Sahifalar o'rtasidagi o'tishlar oddiy `<a>` teglar orqali amalga oshirilgan.
- Rasmlar va aktivlar `assets/` papkasida saqlangan.
